# 🎨 Personnaliser votre Documentation DMS

Bienvenue dans le guide de **personnalisation de la documentation** Dervox DMS. Ce guide vous montre comment styliser vos pages et ajouter des fonctionnalités interactives.

---

## 📁 Organisation des fichiers

Placez vos images dans le dossier `static/img` et vos composants React dans `src/components`.

```bash
📁 docs/
   └── personnaliser-doc.md
📁 static/
   └── img/
       └── logo-dms.png
📁 src/
   └── components/
       └── BoutonAlerte.js
